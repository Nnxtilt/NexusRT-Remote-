ИНСТРУКЦИЯ ПО УСТАНОВКЕ и Настройка: 
1. Загрузите .apk файл и установите его.
2. откройте приложение.
3. выдайте разрешение Accessbility Service.
4. зайдите в Bot Settings.
5. вставьте свой Api token бота (получить можно в @BotFather)
6. вставьте свой Chai ID (получить можно в @Getmyid_bot)
7. далее запустите бота
8. вы также можете выключить Chat ID, но тогда любой сможет управлять вашим устройством.
9. в разделе Permissions вы можете настроить разрешения (переключатели функций).
10. в разделе SHH вы можете блокировать пользователей, и они не смог управлять вашим телефоном, но вы также можете их разблокировать.
