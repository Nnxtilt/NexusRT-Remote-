ИНСТРУКЦИЯ ПО УСТАНОВКЕ и Настройка: 
1. Загрузите .apk файл и установите его.
2. откройте приложение.
3. выдайте разрешение Accessbility Service и разрешение на изменение системных настроек.
5. зайдите в Bot Settings.
6. вставьте свой Api token бота (получить можно в @BotFather)
7. вставьте свой Chai ID (получить можно в @Getmyid_bot)
8. далее запустите бота
9. вы также можете выключить Chat ID, но тогда любой сможет управлять вашим устройством.
10. в разделе Permissions вы можете настроить разрешения (переключатели функций).
11. в разделе SHH вы можете блокировать пользователей, и они не смогут управлять вашим телефоном, но вы также можете их разблокировать.
